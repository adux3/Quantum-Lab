SIMPLE PHP CONSOLE APPLICATION (WINDOWS)

1. RUN. 

C:\Quantum-Lab:\php\php.exe -f "..\demo.php" attr1(command) attr2 attr3 attr4 ...

2. COMMANDS.

2.1. list
Return all people.

2.2. find name
Return people filtered by name.

2.3. languages language1 language2
Return people which knows all that languages.

2.3. addPerson firstName lastName language1 language2 ...
Add new person to database.

2.4. removePerson id
Remove person from database.

2.5. addLanguage language id
Add person language. 

2.6. removeLanguage language id
Remove person language.
